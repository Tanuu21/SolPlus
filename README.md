# 🧠 AI Intelligence Engine

SOLPLUS includes a custom AI-style scoring engine that calculates:

- Rug Risk Score
- Whale Interest Score
- Momentum Score
- Smart Money Score
- Survival Probability
- Hype Score

The platform analyzes:
- holder concentration
- whale activity
- liquidity changes
- transaction velocity
- buy/sell ratios
- token momentum
- developer wallet activity

to generate real-time AI-powered intelligence signals.
